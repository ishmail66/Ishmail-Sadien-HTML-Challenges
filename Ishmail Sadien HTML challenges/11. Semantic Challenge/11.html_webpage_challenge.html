The purpose of a section element is to represent a standalone element on the page. The section can contain headers, documents, videos or anything else the webpage needs to display.
The footer element is used to identify the footer of which ever section you are working in. This can be used in sections, articles, pages and even on docutments. It usually contains the copyrights, identical navigation to the header and publisher/author name.
